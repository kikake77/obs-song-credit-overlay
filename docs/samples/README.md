# TSVインポート用サンプル

`歌枠セットリスト例.tsv` は、Song Credit Manager for OBSの **入出力**タブを試すための説明用ファイルです。

アプリ起動時のチュートリアルには、この123曲から固定抽選した10曲だけが表示されます。全曲を試す場合は、次の手順でこのTSVを読み込んでください。

## 読み込み方

1. Song Credit Manager for OBSを起動します。
2. **入出力**タブを開きます。
3. **CSV／TSV／TXTを読み込む**を押します。
4. このフォルダーの `歌枠セットリスト例.tsv` を選びます。
5. 左のセットリストへ123曲が表示されたら成功です。

TSVはタブ区切りのテキストファイルです。ExcelやGoogleスプレッドシートでも編集できます。

## クレジットについて

作詞・作曲・編曲はMusicBrainzを中心に検索し、一部を公式サイトや楽曲情報で補完した候補です。検索データには不足や誤りがあり得ます。配信で使う前に、CDブックレット、公式サイト、配信サービスの公式クレジットなどでも確認してください。

入力された曲名のうち、明確な表記違いは次のように補正しています。

- `Endress Story` → `Endless Story`
- `secret base` → `secret base ～君がくれたもの～`
- `月陽 ツキアカリ` → `月陽-ツキアカリ-`
- `幽霊船線` → `幽霊船戦`
